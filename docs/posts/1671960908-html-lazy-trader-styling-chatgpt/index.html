You: In plain css, I want the label, input box and the button to appear in a single row in the middle of the screen.

ChatGPT: To create a layout with a label, input box, and button in a single row in the middle of the screen using CSS, you can use the display: flex property to create a flex container, the justify-content: center property to center the elements horizontally, and the align-items: center property to center the elements vertically.

Here’s an example of how you can create this layout in your HTML page:

<style>
  .container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
  }
</style>

<div class="container">
  <label for="input">Input:</label>
  <input type="text" id="input">
  <button>Submit</button>
</div>

This will create a layout with a label, input box, and button in a single row in the middle of the screen. The height: 100vh property sets the height of the container to the full height of the viewport, so the elements will be vertically centered within the viewport.
